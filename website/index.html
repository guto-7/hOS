About

Built for private, on-device diagnostics.

Evidence-Based

Every diagnostic algorithm is peer-reviewed and published. From HOMA-IR and FIB-4 to PhenoAge and Allostatic Load — deterministic evaluations across metabolic, hormonal, and longevity domains. Reproducible and auditable.

Multi-Domain Analysis

Upload blood panels, medical images, or body composition reports. Each diagnostic domain runs as an independent node with its own processing pipeline. A central orchestrator synthesises cross-diagnostic insights.

Private by Design

Your health data never leaves your device. No cloud storage, no user accounts, no telemetry. All analysis runs locally — your diagnostics stay yours.

Privacy

Quick answers

Questions about hOS

Does hOS run locally?

Yes. hOS is designed to run on your device, with on-device analysis as the default experience.

Does my health data get uploaded to the cloud?

No. hOS is designed to keep your health data on your device. All analysis runs locally. Optional network features are always user-controlled.

What files can I upload?

Blood work PDFs from pathology labs, medical images (X-rays, ultrasound, CT screenshots), and BIA body composition reports. Each file type is processed by its corresponding diagnostic node.

Which platforms are supported?

hOS is built for macOS, Windows, and Linux.

Is hOS a replacement for a doctor?

No. hOS is an interpretation and education tool. It should not be used as a substitute for professional medical advice.
